# Personal Portfolio Website

This is my personal portfolio website built using React.js.
The project focuses on frontend development and demonstrates my understanding of React fundamentals, component structure, reusable components, and responsive UI.

## Tech Stack
- React.js
- Tailwind CSS
- JavaScript

## Sections
- About
- Skills
- Projects
- Contact

## Setup Steps
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
